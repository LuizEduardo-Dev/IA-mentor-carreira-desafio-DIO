🚀 PROJETO DE PORTFÓLIO
PROJETO: Task Manager Web App
O QUE FAZER:

Criar uma aplicação web completa onde o usuário pode criar, editar, listar e excluir tarefas.

ENTREGÁVEIS:
Frontend funcional (interface com lista de tarefas)
Backend com API REST (CRUD completo)

💬 ROTEIRO DE ENTREVISTAS
PERGUNTA 1: "Explique o que é uma API REST"

COMO RESPONDER:
"É um padrão de comunicação entre sistemas usando HTTP, onde utilizamos métodos como GET, POST, PUT e DELETE para manipular recursos."

PERGUNTA 2: "Como você resolve um bug?"

COMO RESPONDER:
"Primeiro reproduzo o problema, analiso logs, isolo a causa e aplico a correção validando com testes."

PERGUNTA 3: "O que é async/await?"

COMO RESPONDER:
"É uma forma de lidar com código assíncrono em JavaScript de forma mais legível, evitando callbacks e facilitando o fluxo."

PERGUNTA 4: "Como você organiza seu código?"

COMO RESPONDER:
"Uso separação por responsabilidades (controllers, services, etc.), nomes claros e sigo boas práticas como SOLID quando aplicável."

PERGUNTA 5: "Fale sobre um projeto seu"

COMO RESPONDER:
"Explique o problema, a solução, as tecnologias usadas e os desafios enfrentados, destacando decisões técnicas."
